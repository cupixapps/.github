# Cupix Apps

Cupix 프론트엔드 팀의 GitHub organization입니다.

| Repo | Description |
|------|-------------|
| **cupix** | 프론트엔드 모노레포 (메인 앱) |
| **seoul** | Cupix Design System (`@cupix/ngx-ui`) |
| **prototype** | Seoul 기반 UI 프로토타이핑 |
| **cortex** | 개발 활동 통합 플랫폼 (Epic Digital Twin) |
| **common** | `cupix/libs/common` 읽기 전용 미러 |
| **aps-viewer** | APS (Autodesk) 뷰어 |
| **e57-viewer** | E57 포인트클라우드 뷰어 |
| **powerbi-bridge-sample** | Power BI bridge 샘플 앱 |
